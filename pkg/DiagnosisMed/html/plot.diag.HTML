plot.diag {unknown}R Documentation

Nomogram from 2x2 table diagnostic test

Description

This function draw a simple nomogram from an object created by diagtest or diagtestI

Usage

plot.diag(x, print = F, ...)

Arguments

x An object from diagtest or diagtestI
print If TRUE, plot.diag will print in the output window the table with all pre-test and its correspondig post-test probabilities. Default is FALSE
... Futher generic options for plot function

Details

plot.diag will draw a very simple nomogram as many examples from wikipedia http://en.wikipedia.org/wiki/Nomogram. This is not a generic nomogram as shown in many evidenced based medicine texts, because this one shows only pre-test and post-test variabilities with a fixed likelihood ratio. This likelihood is a statistic from an object created by diagtest or diagtestI. To use it, draw, with a rule, a vertical line from a desired pre-test probability, and to find the corresponding post-test probability, draw a horizontal line from the intersection of the curve and the vertical line.

Author(s)

Pedro Brasil; pedro.brasil@ipec.fiocruz.br

References

JA Knotterus, The Evidence Based Clinical Diagnosis; BMJBooks, 2002

See Also

diagtest,nomogram

Examples

#this will draw a ROC curve
mytest<-diagtestI(364,22,17,211, print=T, plot=T)
#this is different and will draw a nomogram
mytest<-diagtestI(364,22,17,211)
plot(mytest)
#draw a nomogram and print it's table
mytest<-diagtestI(364,22,17,211)
plot(mytest,print=T)
