diagtest {unknown}R Documentation

Diagnostic test accuracy evaluation

Description

Diagtest estimate sensitivity, specificity, predictive values, likelihood ratios, area under ROC curve and other validity measures for binary diagnostic test evaluation.

Usage

diagtest(gold, test1, print = T, plot = F)
diagtestI(TP,FN,FP,TN,print = T, plot = F)

Arguments

gold The reference test (gold stanrd)
test1 The index test (test under evaluation)
TP A number representing True Positives from a 2x2 table
FN A number representing False Negatives from a 2x2 table
FP A number representing False Positives from a 2x2 table
TN A number representing True Negatives from a 2x2 table
print If TRUE, diagtest will print in the output window the statistics resulted from the 2x2 table. Default is TRUE.
plot If TRUE, diagtest will plot a ROC curve of the test under evaluation. This plot may later be edited, as any ohter plot, with title, legends etc. Default is FALSE.

Details

In diagtest, the values entered must be two variables in a dataframe. The first one the gold standard and the second the index test. These two variables must be coded either as numeric - 0 for negative and 1 for a positive test - or with the words "positive" and "negative". Never code the variables as TRUE or FALSE - diagtest will level this categories uncorrectly. diagtestI executes the very same statistics, but here, the values of a 2x2 table are inputed. TP is true positive; TN is true negative; FP is false positive and FN is false negative. All confidence limits rely on binomial distribuition, which does not give result outside [0:1] such as normal distribution or assymptotic theory.

Value

2x2 table, Sample size, Prevalence, Sensitivity, Specificity, Predictive values, Likelihood ratios, Diagnostic odds ratio, Error trade off, Error rate, Accuracy, Area under ROC cruve and confidence intervals.

Examples

# simulating a dataset
gold<-c("negative","positive","negative","negative","positive",
  "negative","positive","negative","negative","positive","negative","positive")
test<-c("negative","negative","negative","positive","positive",
  "negative","positive","negative","negative","positive","negative","positive")
center<-rep(1:3,4)  
data<-cbind(gold,test,center)
for(i in 1:6) {data<-rbind(data,data)}
data<-as.data.frame(data)
data[2,2]<-"positive"
data[12,2]<-"positive"
#attaching dataset
attach(data)
#removing unwanted objects
rm(gold,test,center)
# running the function
mytest<-diagtest(gold,test,print=F)
print(mytest)
# the same as
diagtest(gold,test)
# draw a nomogram from this test
plot(mytest)
#different from - draw a ROC curve from this test
diagtest(gold,test,plot=T,print=F)
#inserting values from a 2x2 table
diagtestI(364,22,17,211)
