print.diag {unknown}R Documentation

Print results from diagtest function

Description

This functions prints in the output window the statistics from diagtest and diatestI functions.

Usage

print.diag(x, ...)

Arguments

x An object created by diagtest or diagtestI.
... Other generic print options.

Author(s)

Pedro Brasil - pedro.brasil@ipec.fiocruz.br

See Also

{diagtest}

Examples

mytest<-diagtestI(364,22,17,211, print=F)
print(mytest)
# the same as 
diagtestI(364,22,17,211)
